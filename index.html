Projeto CCI

Progressive Web Applications (PWA) são aplicações web que se comportam e se assimila com aplicativos móveis e se transformaram em uma alternativa ao aplicativo de código nativo por serem confiáveis, rápidas e envoltórios. Com o emprego dessa tecnologia, o objetivo desse trabalho é construir uma aplicação que consuma os recursos users, post, comentários e todos empregando recursos do Bootrasp 4.
